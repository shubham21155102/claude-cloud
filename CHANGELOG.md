# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-19

### Added
- Initial release of Claude Cloud
- CLI tool for automated GitHub contributions using Claude AI
- `setup` command for initial configuration
- `contribute` command for working on any repository
- `config` command to view current settings
- Support for interactive prompts and command-line options
- Automatic repository cloning and git configuration
- Integration with Claude CLI using `--dangerously-skip-permissions` flag
- Cross-platform support (Linux, macOS, Windows)
- Installation scripts for easy setup
- Comprehensive documentation and examples
- GitHub Actions workflow for testing

### Features
- Universal repository support
- Automatic PR creation with user's name
- Interactive editor for issue descriptions
- Working directory management
- Git user configuration
- Error handling and validation
